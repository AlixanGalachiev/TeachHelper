/classrooms
    POST   /      - класс создаётся(даётся имя)
    GET    /      - получить все классы(по фильтрам - id учителя,)
    PATCH {id}/   - обновить имя класса
    DELETE {id}/  - удалить имя класс
    GET {id}/     - получить класс с его  по фильтрам(id, name) 
    - должен вернутся список содержащий 
        имя ученика
        количество заданных заданий
        его средний балл
        и список работ которые ему задавал(текущий учитель)

/tasks

    GET    /{id}   получить детальную задачи
    UPDATE /{id}   обновить задачу
    DELETE /{id}   удалить задачу и работы связанные

/exercises
    POST   /
    GET    /
    GET    /{id}
    PATCH  /{id}
    DELETE /{id}

Работы создаються учителем и учеником,
    Учитель создаёт задание, выбирает учеников, и задаёт задание(работы создаются)
    Задания может изменяться перед изменением предупреждать учителя
    Ученик может просто создать работу и отправить её учителю.
